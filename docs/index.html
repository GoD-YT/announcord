Announcord

Invite

A simple Discord bot for announcing when users join and leave your channels.

Customizable features including:
  • Announcement behavior
  • Announcement message for enters and exits
  • Various voices and languages to pick from
  • Whitelist/blacklist channels
See usage details for more information on how to customize your announcer
Configure Announcord

Announcord's behavior and settings can be tweaked to fit your needs using the commands below. All commands will need be to preceded by the command prefix (see below).

Note: All commands and options are case-insensitive and can be trimmed down to a minimum of two characters. For commands with similar spelling, you will have to be more specific. For example the prefix command can be reduced to simply pr.

Command Description Usage
prefix Use to change the command prefix. The default is set to ann!. The prefix ann$ is always available as an alternative.
ann!prefix text
announce-style Change the announcement behavior to either only announce channel joins, exits, or both. The default is set to both.

Alternate options are:
  • join to only have joins announced
  • exit to only have exits announced
  • ann!announce-style both
  • ann!announce-style join
  • ann!announce-style exit
ignoreEmpty Tells Announcord to ignore annoucing when a user enters an empty channel or is the last to leave a channel. This behavior is on by default.

See the includeEmpty command to toggle this setting.
ann!ignoreEmpty
includeEmpty Tells Announcord to annouce even when a user enters an empty channel or is the last to leave a channel. This behavior is off by default.

See the ignoreEmpty command to toggle this setting.
ann!includeEmpty
showVoices Sends a direct message containing available voices which can be filtered by providing a language or language code.

The supported languages (codes) are:
  • US English (en-US)
  • British English (en-GB)
  • Australian (en-AU)
  • Indian English (en-IN)
  • Welsh English (en-GB-WLS)
  • US Spanish (es-US)
  • Castilian Spanish (es-ES)
  • Mexican Spanish (es-MX)
  • German (de-DE)
  • Danish (da-DK)
  • Canadian French (fr-CA)
  • French (fr-FR)
  • Icelandic (is-IS)
  • Italian (it-IT)
  • Japanese (jp-JP)
  • Korean (ko-KR)
  • Norwegian (nb-NO)
  • Dutch (nl-NL)
  • Polish (pl-PL)
  • Brazilian Portuguese (pt-BR)
  • Portuguese (pt-PT)
  • Romanian (ro-RO)
  • Russian (ru-RU)
  • Swedish (sv-SE)
  • Turkish (tr-TR)
ann!showVoices
ann!showVoices english
ann!showVoices es-es
voice Change the voice used by Announcord. The default is set to Matthew (en-US).

There are various voice profiles that be used for the supported languages.
  • ann!voice matthew
  • ann!voice astrid
enterAlert Sets the announcement message for when a user enters a channel. The word %name can be used in the message and user's name will be used instead.

The default is set to %name has entered the channel (en-US).
  • ann!enterAlert text
exitAlert Sets the announcement message for when a user exits a channel. The word %name can be used in the message and user's name will be used instead.

The default is set to %name has left the channel (en-US).
  • ann!exitAlert text
whitelist Places the given channel on a whitelist. If there are any channels on the whitelist only these channels will be announced in. By default, the whitelist is empty and all channels will be announced in.

Note:The channel name can be truncated and the first channel found to match will be used.
  • ann!whitelist channel
blacklist Places the given channel on a blacklist. Any channels on the blacklist will not be announced in. By default, the blacklist is empty and all channels will be announced in.

Note:The channel name can be truncated and the first channel found to match will be used.
  • ann!blacklist channel
unlist Removes the channel from either the whitelist or blacklist if it exists on any.

Note:The channel name can be truncated and the first channel found to match will be used.
  • ann!unlist channel
settings Sends an embedded message to the current text channel with the current Announcord configuration
  • ann!settings
resetconfig Reset all Announcord settings to their default values
  • ann!resetconfig
help Sends a direct message with help and command information
  • ann!help
